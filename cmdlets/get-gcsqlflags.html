Get-GcSqlFlags

NAME
    Get-GcSqlFlags
    
SYNOPSIS
    Lists all available database flags for Google Cloud SQL instances.
    
SYNTAX
    
DESCRIPTION
    Lists all available database flags for instances.
    

PARAMETERS
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
OUTPUTS
    Google.Apis.SQLAdmin.v1beta4.Data.Flag
        
    
    
    
    ----------  EXAMPLE 1  ----------
    
    Gets a list of database flags available for instances.
    PS C:\> Get-GcSqlFlags
    If successful, the command returns a list of database flags available for instances.
