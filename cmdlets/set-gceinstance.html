Set-GceInstance

NAME
    Set-GceInstance
    
SYNOPSIS
    Sets various attributes of a VM instance.
    
SYNTAX
    Set-GceInstance [-Project ] [-Zone ] [-Instance]  -NetworkInterface  [-NewAccessConfig ] [-DeleteAccessConfig ] []
    
    Set-GceInstance [-Project ] [-Zone ] [-Instance]  [-AddDisk ] [-DetachDisk ] []
    
    Set-GceInstance [-Project ] [-Zone ] [-Instance]  [-AddMetadata ] [-RemoveMetadata ] []
    
    Set-GceInstance [-Project ] [-Zone ] [-Instance]  [-AddTag ] [-RemoveTag ] []
    
    
DESCRIPTION
    With this cmdlet, you can update metadata, attach and detach disks, add and remove acces configs, or add and remove tags.
