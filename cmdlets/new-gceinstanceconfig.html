New-GceInstanceConfig

NAME
    New-GceInstanceConfig
    
SYNOPSIS
    Makes a new Google Compute Engine VM instance description.
    
SYNTAX
    New-GceInstanceConfig [-Name]  [-CanIpForward ] [-Description ] -Disk  -MachineType  [-Metadata ] [-NetworkInterface ] 
    [-Scheduling ] [-ServiceAccount ] [-Tag ] []
    
    New-GceInstanceConfig [-Name]  [-CanIpForward ] [-Description ] -DiskImage  -MachineType  [-Metadata ] [-NetworkInterface ] 
    [-Scheduling ] [-ServiceAccount ] [-Tag ] []
    
    New-GceInstanceConfig [-Name]  [-CanIpForward ] [-Description ] -DiskSource  -MachineType  [-Metadata ] [-NetworkInterface ] 
    [-Scheduling ] [-ServiceAccount ] [-Tag ] []
    
    
DESCRIPTION
    Makes a new Google Compute Engine VM instance description. Use Add-GceInstance to instantiate the instance.
