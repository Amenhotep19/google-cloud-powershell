New-GceServiceAccountConfig

NAME
    New-GceServiceAccountConfig
    
SYNOPSIS
    Creates a new ServiceAccount object.
    
SYNTAX
    New-GceServiceAccountConfig [-Email] <string> -ScopeUri <string[]> [<CommonParameters>]
    
    New-GceServiceAccountConfig [-Email] <string> [-ScopeUri <string[]>] [-BigQuery <SwitchParameter>] [-BigtableAdmin <BigTableAdminEnum> {None | Tables | Full}] [-BigtableData <ReadWrite> {None | Read | Write | ReadWrite | Full}] [-CloudDatastore <SwitchParameter>] 
    [-CloudLogging <ReadWrite> {None | Read | Write | ReadWrite | Full}] [-CloudMonitoring <ReadWrite> {None | Read | Write | ReadWrite | Full}] [-CloudPubSub <SwitchParameter>] [-CloudSQL <SwitchParameter>] [-Compute <ReadWrite> {None | Read | Write | ReadWrite | 
    Full}] [-ServiceControl <bool>] [-ServiceManagement <bool>] [-Storage <ReadWrite> {None | Read | Write | ReadWrite | Full}] [-TaskQueue <SwitchParameter>] [-UserInfo <SwitchParameter>] [<CommonParameters>]
    
    
DESCRIPTION
    Creates a new ServiceAccount object. These objects are used by New-GceInstanceConfig and Add-GceInstanceTemplate cmdlets to link to service accounts and define scopes. These scopes in turn let your instances access Google Cloud Platform resources.
