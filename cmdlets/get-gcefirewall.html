Get-GceFirewall

NAME
    Get-GceFirewall
    
SYNOPSIS
    Gets firewall rules for a project.
    
SYNTAX
    Get-GceFirewall [-Project ] [[-FirewallName] ] []
    
    
DESCRIPTION
    Gets firewall rules for a project.
    

PARAMETERS

        The Project to get the firewall rule of.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -FirewallName

        Required?                    false

        Accept wildcard characters?  false

        Accept wildcard characters?  false

        The name of the firewall rule to get.
        
        This is an alias of the FirewallName parameter.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue)
