Remove-GcdManagedZone

NAME
    Remove-GcdManagedZone
    
SYNOPSIS
    Removes an existing Google DNS ManagedZone within a Project.
    
SYNTAX
    Remove-GcdManagedZone [-Project <string>] [-Zone] <string> [-Force <SwitchParameter>] [<CommonParameters>]
    
    
DESCRIPTION
    Deletes the specified ManagedZone (and returns nothing).
    
    If a Project is specified, it will instead remove the specified ManagedZone from that project. The optional switch -Force will force removal of even non-empty ManagedZones (e.g., zones with non-NS/SOA type records).
