Set-GceManagedInstanceGroup

NAME
    Set-GceManagedInstanceGroup
    
SYNOPSIS
    Changes the data of a Google Compute Engine instance group manager.
    
SYNTAX
    Set-GceManagedInstanceGroup [-Project ] [-Zone ] [-Name]  -Abandon  -InstanceUri  []
    
    Set-GceManagedInstanceGroup [-Project ] [-Zone ] [-Name]  -Abandon  -InstanceObject  []
    
    Set-GceManagedInstanceGroup [-Project ] [-Zone ] [-Name]  -Delete  -InstanceUri  []
    
    Set-GceManagedInstanceGroup [-Project ] [-Zone ] [-Name]  -Delete  -InstanceObject  []
    
    Set-GceManagedInstanceGroup [-Project ] [-Zone ] [-Name]  -Recreate  -InstanceUri  []
    
    Set-GceManagedInstanceGroup [-Project ] [-Zone ] [-Name]  -Recreate  -InstanceObject  []
    
    Set-GceManagedInstanceGroup [-Project ] [-Zone ] [-Name]  -Size  []
    
    Set-GceManagedInstanceGroup [-Project ] [-Zone ] [-Name]  -Template  []
    
    Set-GceManagedInstanceGroup [-Project ] [-Zone ] [-Name]  -TargetPoolUri  []
    
    
DESCRIPTION
    Changes the data of a Google Compute Engine instance group manager. As a whole, the group can be resized, have its template set, or have its target pools set. Member instances can be abandoned, deleted, or recreated.
