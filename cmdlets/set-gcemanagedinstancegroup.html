Set-GceManagedInstanceGroup

NAME
    Set-GceManagedInstanceGroup
    
SYNOPSIS
    Changes the data of a Google Compute Engine instance group manager.
    
SYNTAX
    Set-GceManagedInstanceGroup [-Project <string>] [-Zone <string>] [-Name] <string> -Abandon <SwitchParameter> -InstanceUri <string[]> [<CommonParameters>]
    
    Set-GceManagedInstanceGroup [-Project <string>] [-Zone <string>] [-Name] <string> -Abandon <SwitchParameter> -InstanceObject <Instance[]> [<CommonParameters>]
    
    Set-GceManagedInstanceGroup [-Project <string>] [-Zone <string>] [-Name] <string> -Delete <SwitchParameter> -InstanceUri <string[]> [<CommonParameters>]
    
    Set-GceManagedInstanceGroup [-Project <string>] [-Zone <string>] [-Name] <string> -Delete <SwitchParameter> -InstanceObject <Instance[]> [<CommonParameters>]
    
    Set-GceManagedInstanceGroup [-Project <string>] [-Zone <string>] [-Name] <string> -Recreate <SwitchParameter> -InstanceUri <string[]> [<CommonParameters>]
    
    Set-GceManagedInstanceGroup [-Project <string>] [-Zone <string>] [-Name] <string> -Recreate <SwitchParameter> -InstanceObject <Instance[]> [<CommonParameters>]
    
    Set-GceManagedInstanceGroup [-Project <string>] [-Zone <string>] [-Name] <string> -Size <int> [<CommonParameters>]
    
    Set-GceManagedInstanceGroup [-Project <string>] [-Zone <string>] [-Name] <string> -Template <string> [<CommonParameters>]
    
    Set-GceManagedInstanceGroup [-Project <string>] [-Zone <string>] [-Name] <string> -TargetPoolUri <string[]> [<CommonParameters>]
    
    
DESCRIPTION
    Changes the data of a Google Compute Engine instance group manager. As a whole, the group can be resized, have its template set, or have its target pools set. Member instances can be abandoned, deleted, or recreated.
