Find-GcsObject

NAME
    Find-GcsObject
    
SYNOPSIS
    Returns all Cloud Storage objects identified by the given prefix string.
    
SYNTAX
    Find-GcsObject [-Bucket]  [[-Prefix] ] [-Delimiter ] []
    
    
DESCRIPTION
    Returns all Cloud Storage objects identified by the given prefix string. If no prefix string is provided, all objects in the bucket are returned.
    
    An optional delimiter may be provided. If used, will return results in a directory-like mode, delimited by the given string. e.g. with objects "1, "2", "subdir/3" and delimited "/"; 
    "subdir/3" would not be returned. (There is no way to just return "subdir" in the previous example.)
    

PARAMETERS
    -Bucket 
        Name of the bucket to search.
        
        Required?                    true
        Position?                    0
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Prefix 
        Object prefix to use. e.g. "/logs/". If not specified all objects in the bucket will be returned.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Delimiter 
        Returns results in a directory-like mode, delimited by the given string. e.g. with objects "1, "2", "subdir/3" and delimited "/", "subdir/3" would not be returned.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
