Find-GcsObject

NAME
    Find-GcsObject
    
SYNOPSIS
    Returns all Cloud Storage objects identified by the given prefix string.
    
SYNTAX
    Find-GcsObject [-Bucket] <string> [[-Prefix] <string>] [-Delimiter <string>] [<CommonParameters>]
    
    
DESCRIPTION
    Returns all Cloud Storage objects identified by the given prefix string. If no prefix string is provided, all objects in the bucket are returned.
    
    An optional delimiter may be provided. If used, will return results in a directory-like mode, delimited by the given string. e.g. with objects "1, "2", "subdir/3" and delimited "/"; "subdir/3" would not be returned. (There is no way to just return "subdir" in the 
    previous example.)
    

PARAMETERS
    -Bucket <string>
        Name of the bucket to search. Will also accept a Bucket object.
        
        Required?                    true
        Position?                    0
        Default value                
        Accept pipeline input?       true (ByValue)
        Accept wildcard characters?  false
        
    -Prefix <string>
        Object prefix to use. e.g. "/logs/". If not specified all objects in the bucket will be returned.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Delimiter <string>
        Returns results in a directory-like mode, delimited by the given string. e.g. with objects "1, "2", "subdir/3" and delimited "/", "subdir/3" would not be returned.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    System.String
        
    
        Name of the bucket to search. Will also accept a Bucket object.
    
    
OUTPUTS
    Google.Apis.Storage.v1.Data.Object
        
    
    
    
    ----------  EXAMPLE 1  ----------
    
    Get all objects in a Storage Bucket
    PS C:\> Find-GcsObject -Bucket "widget-co-logs"
    
    
    
    ----------  EXAMPLE 2  ----------
    
    Get all objects in a specific folder Storage Bucket.
    PS C:\> Find-GcsObject -Bucket "widget-co-logs" -Prefix "pictures/winter" -Delimiter "/"
    Because the Delimiter parameter was set, will not return objects under "pictures/winter/2016/". The search will omit any objects matching the prefix containing the delimiter.
    
    
    
    ----------  EXAMPLE 3  ----------
    
    Get all objects in a specific folder Storage Bucket. Will return objects in pictures/winter/2016/.
    PS C:\> Find-GcsObject -Bucket "widget-co-logs" -Prefix "pictures/winter"
    Because the Delimiter parameter was not set, will return objects under "pictures/winter/2016/".
