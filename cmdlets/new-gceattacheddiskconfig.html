New-GceAttachedDiskConfig

NAME
    New-GceAttachedDiskConfig
    
SYNOPSIS
    Creates a single new AttachedDisk object.
    
SYNTAX
    New-GceAttachedDiskConfig [-Source]  [-AutoDelete ] [-Boot ] [-Nvme ] [-DeviceName ] [-ReadOnly ] []
    
    New-GceAttachedDiskConfig [-SourceImage]  [-Name ] [-DiskType ] [-Size ] [-AutoDelete ] [-Boot ] [-Nvme ] [-DeviceName ] [-ReadOnly ] 
    []
    
    
DESCRIPTION
    Creates a single new AttachedDisk object. These objects are used by New-GceInstanceConfig, Add-GceInstance, and Add-GceInstanceTemplate.
    

PARAMETERS
    -Source 
        The URI of the preexisting disk to attach to an instance.
        
        Required?                    true
        Position?                    0
        Default value                
        Accept pipeline input?       true (ByValue)
        Accept wildcard characters?  false
        
    -SourceImage 
        The source image of the new disk.
        
        Required?                    true
        Position?                    0
        Default value                
        Accept pipeline input?       true (ByValue)
        Accept wildcard characters?  false
        
    -Name 
        The name of the disk to create.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -DiskType 
        Specifies the type of the disk. Defaults to pd-standard.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Size 
        The size of the disk to create, in GB.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -AutoDelete 
        When set, disk will be deleted when the instance is.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Boot 
        When set, describes the boot disk of an instance.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Nvme 
        When set, the disk interface will be NVME rather than SCSI.
        
        Required?                    false
        Position?                    named
        Default value                False

        Accept wildcard characters?  false
        
    -DeviceName 
        The name of the disk on the instance.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ReadOnly 
        Set to limit the instance to read operations.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       false
        Accept wildcard characters?  false
