Get-GceDisk

NAME
    Get-GceDisk
    
SYNOPSIS
    Gets the Google Compute Engine disks associated with a project.
    
SYNTAX
    Get-GceDisk [-Project ] [[-Zone] ] [[-DiskName] ] []
    
    
DESCRIPTION
    Returns the project's Google Compute Engine disk objects.
    
    If a Project is specified, will instead return all disks owned by that project. Filters, such as Zone or Name, can be provided to restrict the objects returned.
    

PARAMETERS

        The project to check for Compute Engine disks.
