Add-GceInstanceTemplate

NAME
    Add-GceInstanceTemplate
    
SYNOPSIS
    Adds an instance template to Google Compute Engine.
    
SYNTAX
    Add-GceInstanceTemplate [-Project ] [-Object]  []
    
    Add-GceInstanceTemplate [-Project ] [-Name]  [-MachineType]  [-CanIpForward ] [-Description ] [-BootDiskImage ] [-ExtraDiskName 
    ] [-Disk ] [-Metadata ] [-Network ] [-NoExternalIp ] [-Preemptible ] [-AutomaticRestart ] 
    [-TerminateOnMaintenance ] [-ServiceAccount ] [-Tag ] []
    
    
DESCRIPTION
    Adds an instance template to Google Compute Engine.
    

PARAMETERS

        The project that will own the instance template.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Object 
        An instance template object to add to Google Compute Engine.
        
        Required?                    true

        Default value

        Accept wildcard characters?  false

        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -MachineType

        Accept wildcard characters?  false
        
    -CanIpForward

        Accept wildcard characters?  false

        Accept pipeline input?       false
        Accept wildcard characters?  false
