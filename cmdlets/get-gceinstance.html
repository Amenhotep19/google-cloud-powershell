Get-GceInstance

NAME
    Get-GceInstance
    
SYNOPSIS
    Gets information about Google Compute Engine VM Instances.
    
SYNTAX
    Get-GceInstance [-Project ] [-SerialPortOutput ] []
    
    Get-GceInstance [-Project ] -Zone  [-SerialPortOutput ] []
    
    Get-GceInstance [-Project ] [-Zone ] [-Name]  [-SerialPortOutput ] []
    
    Get-GceInstance [-Project ] [-Zone ] -ManagedGroupName  [-SerialPortOutput ] []
    
    Get-GceInstance [-Object]  [-SerialPortOutput ] []
    
    Get-GceInstance [-ManagedGroupObject]  [-SerialPortOutput ] []
    
    
DESCRIPTION
    Gets information about VM instances.
    

PARAMETERS

        The project that owns the instances.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false

        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Name

        Required?                    true
        Position?                    0
        Default value                
        Accept pipeline input?       true (ByValue)
        Accept wildcard characters?  false
        
    -Object 
        The Instance object to get a new copy of.
        
        Required?                    true

        Default value

        Accept wildcard characters?  false

        Required?                    true
        Position?                    named

        Accept pipeline input?       false

    -ManagedGroupObject

        Accept wildcard characters?  false

        When this switch is set, the cmdlet will output the string contents of the serial port of the instance rather than the normal data about the instance.
        
        Required?                    false
