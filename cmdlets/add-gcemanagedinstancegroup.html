Add-GceManagedInstanceGroup

NAME
    Add-GceManagedInstanceGroup
    
SYNOPSIS
    Creates a new Google Compute Engine instance group manager.
    
SYNTAX
    Add-GceManagedInstanceGroup [-Project <string>] [-Zone <string>] [-Name] <string> [-InstanceTemplate] <string> [-TargetSize] <int> [-BaseInstanceName <string>] [-Description <string>] [-TargetPool <string[]>] [-PortName <string[]>] [-PortNumber 
    <int[]>] [-NamedPort <NamedPort[]>] [<CommonParameters>]
    
    Add-GceManagedInstanceGroup [-Project <string>] [-Zone <string>] [-Object] <InstanceGroupManager> [<CommonParameters>]
    
    
DESCRIPTION
    Creates a new Google Compute Engine instance group manager.
