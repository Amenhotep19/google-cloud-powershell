Get-GcsObject

NAME
    Get-GcsObject
    
SYNOPSIS
    Get-GcsObject returns the Google Cloud Storage Object metadata with the given name. (Use Find-GcsObject to return multiple objects.)
    
SYNTAX
    Get-GcsObject [-Bucket]  [-ObjectName]  []
    
    
DESCRIPTION
    Returns the give Storage object's metadata.
    

PARAMETERS
    -Bucket 
        Name of the bucket to check.
        
        Required?                    true
        Position?                    0
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ObjectName 
        Name of the object to inspect.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
