Get-GcsObject

NAME
    Get-GcsObject
    
SYNOPSIS
    Get-GcsObject returns the Google Cloud Storage Object metadata with the given name. (Use Find-GcsObject to return multiple objects.)
    
SYNTAX
    Get-GcsObject [-Bucket] <string> [-ObjectName] <string> [<CommonParameters>]
    
    
DESCRIPTION
    Returns the give Storage object's metadata.
    

PARAMETERS
    -Bucket <string>
        Name of the bucket to check. Will also accept a Bucket object.
        
        Required?                    true
        Position?                    0
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ObjectName <string>
        Name of the object to inspect.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
OUTPUTS
    Google.Apis.Storage.v1.Data.Object
        
    
    
    
    ----------  EXAMPLE 1  ----------
    
    Get object metadata.
    PS C:\> Get-GcsObject -Bucket "widget-co-logs" -ObjectName "log-000.txt"
