Google Cloud Storage

All Cloud Storage cmdlets

Google Cloud Storage is neat.

You can do all sorts of stuff with it.

# Use these two commands if you get stuck:
# List all Gcs cmdlets available.
Get-Command | Select-String -Pattern "Gcs"
# Get help about a cmdlet.
Get-Help Get-GcsObject

# List all buckets associated with a project
$project = "gcloud-powershell-testing"
Get-GcsBucket -Project $project | Select -ExpandProperty Name

# Get all objects in a bucket.
$bucket = "chrsmith-debugging"
# Get all objects in a GCS bucket.
Find-GcsObject -Bucket $bucket | Select -ExpandProperty Name

# Write a GCS object to disk.
Read-GcsObject "bucket-name" "object-name" -OutFile "output-file.xxx"
