Write-GcsObject

NAME
    Write-GcsObject
    
SYNOPSIS
    Replaces the contents of a Cloud Storage object.
    
SYNTAX
    Write-GcsObject [-Bucket]  [-ObjectName]  [[-Contents] ] [-Force ] []
    
    Write-GcsObject [-Bucket]  [-ObjectName]  [[-File] ] [-Force ] []
    
    
DESCRIPTION
    Replaces the contents of a Cloud Storage object with data from the local disk or a value from the pipeline.
    

PARAMETERS
    -Bucket 
        Name of the bucket containing the object.
        
        Required?                    true
        Position?                    0
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ObjectName 
        Name of the object to write to.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Contents 
        Text content to write to the Storage object. Ignored if File is specified.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByValue)
        Accept wildcard characters?  false
        
    -File 
        Local file path to read, writing its contents into Cloud Storage.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Force 
        Force the operation to succeed, ignoring errors if no existing Storage object exists.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       false
        Accept wildcard characters?  false

        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    System.String
        Text content to write to the Storage object. Ignored if File is specified.
