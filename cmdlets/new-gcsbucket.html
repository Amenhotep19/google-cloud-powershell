New-GcsBucket

NAME
    New-GcsBucket
    
SYNOPSIS
    Creates a new Google Cloud Storage bucket.
    
SYNTAX
    New-GcsBucket [-Name]  [-Project ] [-StorageClass ] [-Location ] []
    
    
DESCRIPTION
    Creates a new Google Cloud Storage bucket.
    

PARAMETERS

        Required?                    true
        Position?                    0

        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false

        Location for the bucket. e.g. ASIA, EU, US.
